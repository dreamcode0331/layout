  • 5. grid

    BOX 1
    width 유동
    BOX 2
    width 고정

    - box_wrap(컨테이너) : display: grid 선언 후 grid-template-columns 속성을 사용하여 box 2 고정값을 부여하고 box 1은 가변 너비를 갖도록 1fr로 적용한다.

    .layout_05 .box_wrap { display: grid; grid-template-columns: 1fr 684px }

    - 크로스브라우징 이슈 있음
    - display: grid - IE(ver.10~11)에서 부분적으로 지원가능 함.

    BOX 1
    width 유동
    BOX 2
    width 유동

    - 열의 배치를 결정하는 grid-template-columns은 속성을 이용하여 그리드 트랙 크기를 지정한다. 가로너비가 가변크기 이므로 숫자 비율대로 크기를 나누는 단위인 fr(fraction)을 사용한다. box 1 column은 1fr, box 2 column은 minmax(0, 2fr)을 적용한다.

    .layout_05 .box_wrap_02 { display: grid; grid-template-columns: 1fr 2fr}

    - 크로스브라우징 이슈 있음
    - display: grid - IE(ver.10~11)에서 부분적으로 지원가능 함.
