Naming lines when defining a grid (그리드 정의 시, 라인 이름 만들기)

.grid_container { display: grid; grid-template-columns: [main-start] 1fr [content-start] 1fr [content-end] 1fr [main-end]; grid-template-rows: [main-start] 100px [content-start] 100px [content-end] 100px [main-end]; }
.grid_item1 { display: grid; grid-column-start: main-start; grid-row-start: main-start; grid-row-end: main-end; }
.grid_item2 { display: grid; grid-column-start: content-end; grid-row-start: main-start; grid-row-end: content-end; }
.grid_item3 { display: grid; grid-column-start: content-start; grid-row-start: main-start; }

Implicit grid areas from named lines (이름이 주어진 라인으로 만들어지는 잠재적 그리드 영역)

.grid_container { display: grid; grid-template-columns: [main-start] 1fr [content-start] 1fr [content-end] 1fr [main-end]; grid-template-rows: [main-start] 100px [content-start] 100px [content-end] 100px [main-end]; }
.thing { grid-area: content; }
I am placed in an area named content.

Implicit grid areas from named lines (이름이 주어진 영역으로 만들어지는 잠재적 그리드 라인)

.grid_container { display: grid; grid-template-columns: repeat(9, 1fr); grid-auto-rows: minmax(100px, auto); grid-template-areas:
"hd hd hd hd hd hd hd hd hd"
"sd sd sd main main main main main main"
"ft ft ft ft ft ft ft ft ft"; }
.grid_header { grid-area: hd; }
.grid_footer { grid-area: ft; }
.grid_content { grid-area: main; }
.grid_sidebar { grid-area: sd; }

Multiple lines with the same name with repeat() (repeat()을 써서 같은 이름을 가진 여러 개의 라인 만들기)

.grid_container { display: grid; /* grid-template-columns: 12개의 열로 구성된 그리드, [라인 이름] 열트랙 너비 */ grid-template-columns: repeat(12, [col-start] 1fr) }
.grid_item1 { grid-column:col-start / col-start 5 }
.grid_item2 { grid-column:col-start 7 / span 3 /* 7번째 라인부터 3개의 라인에 걸치도록 배치 */ }
I am placed from col-start line 1 to col-start 5
I am placed from col-start line 7 spanning 3 lines
.grid_container { display: grid; grid-template-columns: repeat(12, [col-start] 1fr); grid-gap: 10px; }
.grid_item1 { grid-column: col1-start / col2-start 2; }
.grid_item2 { grid-row: 2; grid-column: col1-start 2 / span 2 col1-start; }
I am placed from col1-start line 1 to col2-start line 2
I am placed from col1-start line 2 spanning 2 lines named col1-start
