Naming a grid area (그리드 영역 이름 지정)

.grid_container { display: grid; grid-template-columns: repeat(9, 1fr); grid-auto-rows: minmax(100px, auto); grid-template-areas:
"hd hd hd hd hd hd hd hd hd"
"sd sd sd main main main main main main"
"ft ft ft ft ft ft ft ft ft"; }
.grid_header { grid-area: hd; }
.grid_footer { grid-area: ft; }
.grid_content { grid-area: main; }
.grid_sidebar { grid-area: sd; }

Using grid-template-areas for UI elements (grid-template-areas UI 요소에 사용)

.grid_container .media { display: grid; grid-template-columns: 1fr 3fr; grid-template-areas: "img content"; }
.grid_container .media .image { grid-area: img; }
.grid_container .media .text { grid-area: content; }
This is a media object example. We can use grid-template-areas to switch around the image and text part of the media object.

Grid definition shorthands (그리드 정의 줄여쓰기)

/* grid-template 은 다음 속성을 설정한다
- grid-template-rows
- grid-template-columns
- grid-template-areas

- 다음 코드는 grid_ex1 예제의 레이아웃과 동일한 레이아웃을 구성한다. */ .grid_container .media { display: grid; grid-template:
"hd hd hd hd hd hd hd hd hd" minmax(100px, auto)
"sd sd sd main main main main main main" minmax(100px, auto)
"ft ft ft ft ft ft ft ft ft" minmax(100px, auto)
/ 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr; }

암시적 그리드 줄여쓰기

/* grid는 암시적 그리드에서 사용되는 속성도 설정한다
- grid-template-rows
- grid-template-columns
- grid-template-areas
- grid-auto-rows
- grid-auto-columns
- grid-auto-flow

- grid-template 줄여쓰기와 똑같은 방법으로 사용할 수 있지만, 해당 속성에 의해 설정했던 다른 속성값들이 재설정 될 수 있음을 유의. */ .grid_container .media { display: grid; grid:
"hd hd hd hd hd hd hd hd hd" minmax(100px, auto)
"sd sd sd main main main main main main" minmax(100px, auto)
"ft ft ft ft ft ft ft ft ft" minmax(100px, auto)
/ 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr; }
